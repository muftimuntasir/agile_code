POS - Point of Sale Credit Limit Odoo Apps

Allow Customer Credit Limit on POS Odoo Apps

POS - Point of Sale Credit Limit Odoo App helps users to apply customer base credit limit on pos order, and able to use assigned credit amount as payment on pos using credit journal. In order to create a credit journal, users can tick the "Is Credit Journal" checkbox on journal view. Users can tick on allow credit checkbox for particular customer and also set amount to raise warning and block credit order. Display warning message. If order amount is exceeding warning limit than also you can validate order and if order amount is exceeding blocking limit than you cannot validate order. Users can see a warning message when a customer is not set , not sufficient credit for payment as well as when payment amount is exceeded.

Features

Allow Credit for the Customer

User can check or uncheck credit for particular customer.

Set Amount to Raise Warning and Block Credit Order

User can set amount to raise warning message and block credit order.

Display Warning Message if Credit is Exceeding the Limit

You can see warning message display ,If order amount is exceeding warning limit amount but you can validate order.

Display Blocking Message if Credit Limit Exceeding

You can see blocking message display ,If order amount is exceeding blocking limit amount and you can't validate order.

Display Credit Limit for the Customers in POS

User can see credit limit for the customers in POS.

Raise Warning/Validation Message

User can see raise warning message, if Unknown Customer, Not Sufficient Credit and Payment Amount Exceeded.
